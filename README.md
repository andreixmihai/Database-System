# Library App

A web application developed using **Python Flask** and **MySQL** for managing a library database.

The application allows users to manage authors, books, and loans through a simple web interface. It demonstrates database relationships, CRUD operations, and backend web development using Flask.

## Features
- Add, edit and delete authors
- Manage books and their availability
- Register book loans
- Track borrowed and returned books
- Relational database using MySQL

## Technologies Used
- Python
- Flask
- MySQL
- HTML (Jinja templates)
- SQL

## Database Structure
The system includes the following tables:
- **authors** – stores author information
- **books** – stores book details
- **members** – library users
- **loans** – borrowing records
